# 抓取指南：怎么拿到番茄前 3 章正文

> 这里的步骤都是**实测跑通过**的（2026-07）。番茄网页会变，若某步失效，按「排错」调整。

## 为什么必须用浏览器

番茄网页正文用了**自定义字体加密**：HTML 里的正文是打乱的私有区码位（形如 ``），
靠一个自定义 web 字体才映射回正确字形。所以：

- ❌ 直接 `curl` / 爬 HTML 拿到的正文是**乱码**。
- ❌ 网页阅读器容器带 `noselect`，**没法直接选中复制**。
- ✅ **真实浏览器渲染后是干净明文**（人眼看到的就是对的）→ 所以我们**截图 + 视觉读字**，天然绕过加密。

> 例外：书籍**元数据**（书名/作者/简介/标签/字数/在读数）在页面里是**明文 JSON**，可以直接读到，不受加密影响。

---

## 主路径（有 claude-in-chrome 浏览器时）

### A. 按书名定位书籍

1. 先建/取标签页：`tabs_context_mcp{ createIfEmpty: true }`。
2. **直接导航到搜索结果页**（最稳，不用跟输入框较劲）：
   ```
   https://fanqienovel.com/search/{书名}
   ```
   例：`https://fanqienovel.com/search/末日全球海洋`
3. 截图 / `read_page` 读结果列表。结果项含：书名、作者、完结状态、分类标签、字数、在读人数、简介、最近更新章节。
4. **认出目标书**：书名+作者对上；同名/仿名很多，务必核对。记下它的元数据备用。

> 备选：若结果不理想，可用搜索引擎兜底——搜「书名 番茄小说 fanqienovel.com」，
> 从结果里拿到 `fanqienovel.com/page/{bookId}` 直接导航。

### B. 打开正文（第 1 章）

- 在搜索结果里点该书的**「立即阅读」**（会新开一个标签页，注意切到新 tabId）；
- 或如果已知章节地址，直接导航 `https://fanqienovel.com/reader/{chapterId}`。

进阅读器后截图确认：顶部有章节标题（如「第1章 xxx」），正文是**可读中文**即成功。

### C. 逐章读取前 3 章

对第 1、2、3 章，每章：
1. 从章首开始截图；`scroll` 向下，继续截图，直到本章正文结束（出现「下一章」按钮或页脚）。
   一章约 2000 字，通常 3–5 屏。
2. 把该章所有截图用视觉**通读并在脑中拼成完整正文**（不必逐字誊写给用户，理解内容即可）。
3. 翻下一章：`find{ query: "下一章" }` 找到「下一章」按钮 → `left_click` 它，
   页面会切到下一章（标题和 URL 都会变）。重复到第 3 章。

> 小技巧：也可从书籍页拿到有序章节 `itemId` 列表，直接导航 `reader/{itemId}` 逐章跳，
> 但「下一章」按钮更简单可靠，优先用它。

### D. 收集元数据

从搜索结果页或书籍页 `https://fanqienovel.com/page/{bookId}` 读取并记录：
书名、作者、完结状态、分类/标签、总字数、在读/阅读人数、官方简介。写进报告抬头。

## 排错

| 现象 | 处理 |
|---|---|
| 首页搜索框输入后不跳转 | 别用输入框，直接导航 `fanqienovel.com/search/{书名}` |
| 「立即阅读」开了新标签页 | 切到新的 tabId 再截图（`tabs_context_mcp` 看当前所有 tab） |
| 正文是方块/乱码 | 字体没加载完，`wait` 1–2 秒后重截；或刷新阅读器页 |
| 出现登录墙 / 章节要付费 | 前几章通常免费；若被挡，截下能看到的部分，如实告诉书友"只抓到前 N 章/半章" |
| 搜不到书 | 让书友确认书名，或直接给番茄 App 的**分享链接**（内含 bookId） |
| 分享链接是短链/App 链接 | 导航过去让它跳转，从最终 URL 里取 `page/{id}` 或 `reader/{id}` |

## 兜底路径（没有浏览器时）

如果环境里没有 claude-in-chrome（或连不上），退化为「书友提供内容」：

1. 让书友发**番茄链接**（书页或分享链接）——至少能拿到明文元数据（书名/作者/简介/标签）做背景。
2. 让书友**截图前 3 章**丢进来——视觉读图，效果和主路径一样。
3. 或让书友**粘贴前 3 章文本**（若他手上有干净文本）。

只要能拿到前 3 章内容，四位品鉴官照常工作。抓不到正文、只有简介时，明确说明"仅凭简介做初步预判，非正式品鉴"。
