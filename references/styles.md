# 文风词典（文风鉴定师用）

给一本书的文风贴标签、描述读起来的体感。文风无绝对好坏，但**要说清它是什么味道、适合什么人**。

## 从几个维度打标签

### 1. 节奏
- `节奏明快`：进展快、不拖，一章一个推进
- `紧凑干净`：信息利落，废话少
- `略拖沓` / `注水严重`：反复内心戏、重复交代、凑字数、一件小事写好几章
- `慢热`：铺垫长，后期才起势（要区分"慢热"和"拖沓"）

### 2. 句式
- `短句爽文体`：短句、分段密、爽点直给（番茄主流）
- `长句/文艺腔`：长句、修辞多、偏文学
- `口语化对话流`：靠大量对话推进
- `复读机`：同类句式/口号反复出现

### 3. 语言质地
- `大白话`：通俗直给，几乎无门槛（不贬义，番茄主流）
- `考究有文采`：用词讲究、有画面感、比喻新
- `口水/粗糙`：用词贫乏、错别字多、病句、AI 味
- `梗多/网感强`：玩梗、沙雕、弹幕感

### 4. 氛围基调
- `轻松沙雕` / `爽感强` / `搞笑`
- `甜宠/工业糖精`
- `严肃/压抑/致郁`
- `悬疑/神秘/诡异`
- `热血/中二`
- `细腻/情感向`

### 5. 其他体感开关
- **信息密度**：高（每段有料）/ 低（水）
- **对话占比**：对话多 vs 叙述多
- **内心戏**：内心独白是"有信息"还是"凑字数"
- **描写**：环境/感官描写是"有画面"还是"套话堆砌"
- **代入感**：主角视角是否让人代入

---

## 常见「毒点」信号（要专门揪出来）

- **工业糖精**：为甜而甜，强行发糖，脱离逻辑
- **尬吹/口号式打脸**："全场哗然""这怎么可能""他到底是什么人"复读
- **强行降智**：反派/配角为衬托主角而智商掉线
- **注水**：一件小事反复描写、大段无信息内心戏、"欲知后事"式凑字
- **信息轰炸**：开局甩一大段设定/名词，劝退
- **人设崩/前后矛盾**：人物性格/设定自打脸
- **三观预警**：过度爹味、恋爱脑、暴力/歧视被当爽点
- **AI 味**：见 `ai-tells.md`（泛化、模板、无细节）

## 输出建议

- 给 3–5 个标签（可正可负），例如：`节奏明快 + 大白话 + 爽感强 + 略中二`。
- 用 2–3 句话描述"读起来像什么"，各引一处原文佐证最突出的**优点**和最明显的**毒点**。
- `文笔质感` 分数：综合语言质地、信息密度、有无毒点。番茄爽文的"大白话"不扣分，
  真正扣分的是**口水/病句/AI 味/严重注水**。耐读、有画面、有个人风格的加分。
