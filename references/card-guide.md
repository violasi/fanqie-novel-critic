# 品鉴卡渲染指南（社交卡片 / 小红书风格）

品鉴文字报告出完后，**再渲染一张赏心悦目的社交卡片**作为成品，方便书友保存、分享、发小红书。

**模板 = [`examples/sample-card.html`](../examples/sample-card.html)**（一份完整、可直接改的实现）。
做法：复制它，把里面的《末日全球海洋》数据换成当前这本书的即可。不要从零手搓，照着改最稳。

## 设计基调（别改坏了）
- 配色：番茄植物色——熟果红 `#E23A26`（主）＋ 藤叶绿 `#2F7A4F`（结构/标签）＋ 籽墨黑 `#241310`，暖纸底。
- 形态：竖版「品鉴票据」——票根穿孔上边 + 标题打头 + 标题下方手盖**评级印章** + 四维评分条 + 四位品鉴官行 + 雷点 + 适合/避雷 + 页脚。
- 不用衬线奶油陶土那套；保持扁平、无渐变滥用。

## 要替换的字段
| 位置 | 内容 |
|---|---|
| `.title` | 书名 |
| `.stamp b` | 评级印章文字（见下方映射） |
| `.meta` 的 `.chip` | 作者/状态/标签/字数/在读（`.chip.k` 那颗给"完结状态"上色） |
| `.band p` | 一句话结论 |
| 四个 `.g` 块 | 每个：`data-v`=分值，`data-good`=好坏分（见下），`.name`=维度名，`越高/越低越好`，`.val`=分值 |
| 四个 `.row` | 每位品鉴官：图标、名称、彩色分数胶囊、一句点评 |
| `.warn ul` | 雷点预警条目 |
| `.aud` | 适合 / 避雷各一句 |

## 评级印章映射（`.stamp`）
| 主编评级 | 印章文字 | 印章颜色（改 `.stamp` 的 border/color） |
|---|---|---|
| 🏆 值得追 | `值得追` | 藤叶绿 `--vine-deep`（默认） |
| 👍 可以追 | `可以追` | 藤叶绿 `--vine-deep`（默认） |
| 😐 食之无味 | `食之无味` | 暖灰 `#8A7E76` |
| ⛔ 建议劝退 | `劝退` | 番茄红 `--tomato-deep` |

## 评分条颜色（自动，别手动填色）
卡片底部脚本按每条的 `data-good`（0–100，越高越好）自动上色：
`≥67 绿 / 40–66 琥珀 / <40 红`。所以你只需算出每个维度的"好坏分"填进 `data-good`：
- **越高越好**的维度（文笔质感、追读指数）：`data-good = 分值`。
- **越低越好**的维度（AI 嫌疑度、套路浓度）：`data-good = 100 − 分值`。

`data-v` 永远填**原始分值**（决定条的长度）；`data-good` 只决定颜色。
品鉴官行里的分数胶囊颜色，同理手动选 好绿/中琥珀/差红。

## 交付
- 存成 `{书名}-品鉴卡.html`。
- 有 Artifact / 网页托管能力时，发布成链接给书友；否则让书友本地打开或截图分享。
- 卡片是**文字报告的补充**，不替代报告；报告给依据，卡片给"赏心悦目的结论"。
