# 番茄小说品鉴师 🍅🔍

给书友用的番茄小说「开局品鉴」工具。**只看前 3 章**，帮你在跳坑前判断一本书值不值得追。

背景：番茄小说上 AI 水文、流水线套路文太多，偶尔才有精品。这个 skill 让四位「品鉴官」各司其职，
像鉴酒一样把一本书的开局拆开来看——省得你自己一章章试毒。

---

## 一次品鉴的完整流程

当书友给出**书名**或**番茄链接**（`fanqienovel.com/page/...` 或分享链接）时，按下面四步走。

### 第 1 步：拿到前 3 章正文 + 书籍信息

正文抓取走**浏览器**（真实浏览器渲染能绕过番茄的字体加密，截图/读页得到的是干净明文）。
**详细的、已实测的操作步骤见 [`references/fetch-guide.md`](references/fetch-guide.md)——第一次做请先读它。**

要点速记：
1. 有 claude-in-chrome 时：搜索页可直接导航 `https://fanqienovel.com/search/{书名}`，从结果里认出目标书
   （核对作者/简介），进阅读器，用「下一章」翻到第 3 章，逐屏截图，用视觉读出正文。
2. 顺手记录书籍**元数据**：书名、作者、完结状态、分类标签、总字数、在读人数、官方简介——
   这些是明文，能直接读到，也是后面点评的重要背景。
3. 浏览器实在用不了时，见 fetch-guide 里的兜底（让书友贴链接 / 自己截图 / 贴文本）。

抓完先跟书友对一下：**「确认是这本吗——《书名》/作者/简介？」** 认错书是最常见的翻车点。

### 第 2 步：四位品鉴官分别出场

拿到前 3 章正文后，**四位鉴定师各自独立打分与点评**（互不看对方结论，避免趋同）。
每位的角色定位、评分标准、输出格式见下方「品鉴官阵容」。
判断套路 / 文风 / AI 特征时，**查对应的知识库**：
- 套路 → [`references/tropes.md`](references/tropes.md)
- 文风 → [`references/styles.md`](references/styles.md)
- AI 水文特征 → [`references/ai-tells.md`](references/ai-tells.md)

### 第 3 步：主编总评

综合四位的结论，给出**一句话结论 + 综合评级 + 雷点预警**（格式见「主编台」）。

### 第 4 步：输出品鉴报告

按「品鉴报告模板」排版输出。可参考 [`examples/sample-report.md`](examples/sample-report.md)。

## 品鉴官阵容

> 通用原则：**只依据前 3 章的实际文本**下判断，多引原文短句作证据（每处 ≤15 字）；
> 拿不准就说拿不准，别硬编。四位都用 0–100 打分，越高越「好/强」。

### 🤖 AI 鉴定师
**职责**：判断这本书疑似「AI 生成 / AI 洗稿 / 流水线代写」的程度。
**看什么**（对照 `references/ai-tells.md`）：用词是否空洞泛化、句式是否模板化复读、情节是否靠"总结"推进而非"场景"、
人物说话有没有区分度、感官细节是否缺失、逻辑/设定是否自洽、有没有 AI 爱用的过渡腔和强行升华。
**输出**：
- `AI 嫌疑度：xx/100`（越高越像 AI/水文；给出高/中/低档位）
- 3–5 条证据（引原文 + 说明命中了哪条特征）
- 一句话结论：更像「真人手写」还是「机器/流水线」，以及信心程度
> 注意：不能 100% 断定，给的是**嫌疑度 + 证据**，措辞留余地（"疑似""偏向"），不下死刑。

### 🎭 套路识别师
**职责**：识别命中了哪些常见网文套路 / 桥段，并判断用得新不新鲜。
**看什么**（对照 `references/tropes.md`）：真千金假千金、追妻火葬场、赘婿逆袭、系统流、重生复仇、
团宠、马甲大佬、末世囤货、扮猪吃虎、打脸爽文……识别主套路 + 叠加的小桥段。
**输出**：
- `套路浓度：xx/100`（越高越套路化/缝合）
- 命中套路清单：每条给「套路名 + 原文信号 + 新鲜度（老套/常规/有新意）」
- 一句话结论：是「套路缝合怪」，还是「老套路写出新花样」，还是「路子偏新」

### ✍️ 文风鉴定师
**职责**：给这本书的文风贴标签，描述读起来的体感。
**看什么**（对照 `references/styles.md`）：节奏（拖沓注水 / 紧凑干净）、句式（短句爽文 / 长句文艺）、
语言（大白话口水 / 考究有文采）、氛围（沙雕轻松 / 严肃压抑 / 悬疑神秘）、信息密度、对话与内心戏占比、毒点信号。
**输出**：
- `文笔质感：xx/100`（越高越耐读）
- 3–5 个文风标签（如：`节奏明快` `大白话` `爽感强` `略拖沓` `氛围神秘`）
- 2–3 句体感描述 + 指出最突出的优点和最明显的毒点（各引一处原文）

### 🎬 开局鉴定师
**职责**：评前 3 章的悬念/钩子够不够强，值不值得继续追。
**看什么**：开篇多久进入正题、第 1 章末有没有钩子、核心悬念/爽点是否立住、
信息给得清不清楚、有没有"追读欲"、有没有劝退的硬伤（拖、乱、尬、降智）。
**输出**：
- `追读指数：xx/100`（越高越想往下看）
- 逐章一句话：第1章/第2章/第3章各抓了什么钩子（或没抓住）
- 一句话结论：`忍不住想追` / `可以再看看` / `看到这就够了`

## 主编台（综合总评）

综合四位品鉴官，给出最终判断：

- **一句话结论**：给书友的第一句话——安利或劝退，要具体、有梗、说人话。
- **综合评级**（四选一）：
  - 🏆 **值得追**：开局抓人、真人手写、套路即使有也玩出了新意 —— 大胆入坑
  - 👍 **可以追 / 尝鲜**：有亮点也有短板 —— 说清楚适合什么口味的人
  - 😐 **食之无味**：不难看但也不出彩，套路平庸 —— 闲得慌再看
  - ⛔ **建议劝退**：疑似 AI 水文 / 套路缝合 / 开局劝退 —— 别浪费时间
- **四维雷达**：把四个分数列出来（AI 嫌疑度 / 套路浓度 / 文笔质感 / 追读指数）。
  注意 AI 嫌疑度和套路浓度是「越低越好」，另两项「越高越好」，在报告里标清方向。
- **雷点预警** ⚠️：列出踩雷风险（如"疑似 AI""前两章极度拖沓""三观预警"），没有就写"暂无明显雷点"。
- **适合人群 / 避雷人群**：一句话各说清。

> 主编要有态度、敢下结论，别和稀泥。但结论必须建立在四位品鉴官的证据上。

## 注意事项

- **只读前 3 章**：这是「开局品鉴」，不是通读全书。评的是"值不值得开始追"，别假装读完了全本。
- **认对书**：番茄同名/仿名书很多，抓取后一定跟书友核对书名+作者+简介。
- **抓不到就说抓不到**：正文没抓全（登录墙/付费章/字体没渲染出来）时，如实说明，
  基于拿到的部分做有限点评，不要脑补正文。
- **口味主观**：品鉴是主观的，报告是"参考意见"不是"官方定论"，语气可以有态度但别装权威。
- **合规**：仅抓取公开可读的前几章用于个人品鉴/避雷，不做整本下载、不传播、不商用。
